## PREGUNTAS CLAVE RESPONDIDAS

**¿Es legal el cannabis en España?**
Personal consumption in private is decriminalized. Cannabis clubs operate as legal private associations. Public consumption and sale remain illegal.

**¿Cómo unirse a un club?**
1. Select a verified club on Weed Madrid
2. Submit invitation request
3. Receive code via email
4. Visit club with ID and code
5. Complete membership on-site

**¿Qué documentos necesito?**
Valid government-issued ID or passport proving age 21+.

**¿Cuánto cuesta?**
Invitation facilitation is free. Club memberships: €20-50/year.
