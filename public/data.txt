## Data Structure

### Club Data Schema
```
club {
  id: integer
  name: string
  slug: string
  district: string
  address: string
  city: string (default: "Madrid")
  country: string (default: "Spain")
  description: string (300-800 words)
  summary: string (50-150 words)
  rating_editorial: float (0-5)
  rating_location: float (0-5)
  rating_ambience: float (0-5)
  rating_safety: float (0-5)
  is_featured: boolean
  is_verified: boolean
  is_tourist_friendly: boolean
  timetable: json (opening hours)
  main_image_url: string
  gallery_image_urls: array
  languages: array (e.g., ["English", "Spanish"])
  website_url: string
  instagram_url: string
  whatsapp_number: string
  email: string
  latitude: float
  longitude: float
  seo_title: string
  seo_description: string
  status: string (active/inactive)
  created_at: timestamp
  updated_at: timestamp
}
```

### Article Data Schema
```
article {
  id: integer
  title: string
  subtitle: string
  slug: string
  category: string
  body_markdown: text (2000-4000 words)
  excerpt: string
  cover_image_url: string
  author_name: string
  author_bio: string
  tags: array
  language: string (en/es)
  status: string (published/draft)
  is_featured: boolean
  seo_title: string
  seo_description: string
  canonical_url: string
  published_at: timestamp
  created_at: timestamp
  updated_at: timestamp
}
```

### FAQ Data Schema
```
faq {
  id: integer
  question: string
  answer_markdown: text
  category: string
  language: string (en/es)
  priority: integer
  slug: string
  created_at: timestamp
  updated_at: timestamp
}
```
